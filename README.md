# ASLDVSCTL
ASLDVSCTL — A modular control and automation framework for AllStarLink 3 and DVSwitch supporting DMR, NXDN, P25, YSF, and D-Star

ASLDVSCTL is a modular control and automation framework for
AllStarLink 3 and DVSwitch.

Features

- DMR
- NXDN
- P25
- YSF
- D-Star
- Profile management
- DTMF control
- Event framework
- State management
- Diagnostics

Status

Development in progress.
